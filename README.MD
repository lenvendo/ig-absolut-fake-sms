
## Описание

Сообщения на отправку извлекаются из nats (subject конфигурируется)

Формат сообщений:
```json
{
    "phone": "+7(999) 999-99-99",
    "code": 1234
}
```

Логи отправки сохраняются в PostgreSQL

## Конфигурация

Создать в корне конфигурационный файл .env

Пример: [.env.example](.env.example)

## Запуск
```
cd structure
docker-compose up 
```
